# LMS React (Vite + Ant Design) 

Link Demo & Source Code

Vercel: https://lms-project-hazel.vercel.app/
GitHub Repository: https://github.com/duytan22/lms-project
## Setup
1. git clone https://github.com/duytan22/lms-project
2. Run `npm install`.
3. Run `npm run dev`.
4. Open http://localhost:5173

## Features
✔ Login:
-Kiểm tra email và password.
✔ Course List
-Hiển thị danh sách.
-Lọc tìm kiếm theo tên, danh mục và theo Level.
-Nút chuyển trang
✔ Add Course
-Form nhập liệu.
-Lưu localStorage.
✔ Edit Course
-Hiển thị lỗi đúng ô sai.
-nút 'exit' để thoát.
✔ Bảo vệ route (ProtectedRoute)
-Nếu chưa login sẽ redirect về /
