# LMS React (Vite + Ant Design) - Sample


## Setup
1. Copy files into a new folder (e.g. `lms-app`).
2. Run `npm install`.
3. Run `npm run dev`.
4. Open http://localhost:5173


## Features
- Login (hardcoded demo user)
- Route protection using Context
- Course List with Ant Design Table (search, filter, pagination)
- Add / Edit course screens using Ant Design Form
- Local state persistence using localStorage
